# Практичне заняття  
## «Робота з Dockerfile та Docker Compose»

### **Мета роботи**
#### Навчитися працювати з директивами Dockerfile і запускати багатосервісний застосунок (Flask + Redis) за допомогою Docker Compose.
## Хіж роботи
### Крок 1 - створення та налаштування ```app.py```
### Крок 2 - створення та налаштування ```Dockerfile```
### Крок 3 - Створення Compose конфігурації ```docker‑compose.yml```
### Крок 4 - Запуск через Compose
### за допомогою команди ```docker compose up --build```
### Крок 5 - Зупинка та очищення
### за допомогою команди ```docker compose down```
